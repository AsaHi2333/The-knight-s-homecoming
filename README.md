# 骑士归家记 demo
2024吉比特高校GameJam游戏通关奖作品
游戏介绍：一位思乡的小骑士在睡梦中被同乡的一位无头骑士前辈托梦，获得了特殊的能力，于是，小骑士决定凭借这个能力回归家乡。
核心玩法为：角色行走时默认进行匀加速，场景中有根据玩家触碰时的水平速度或垂直速度而呈现不同特性的组件，玩家需根据地形限制控制速度，利用这些组件进行闯关，详细教学见demo演示视频
键位：
**WASD移动**  
**1 2 3切换关卡**  
**R重新开始本关卡**  
**到达终点时，按F进入下一关**
